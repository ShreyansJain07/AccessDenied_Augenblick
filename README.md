## Step 1: Install Python Packages

First, you need to install the required Python packages. Open your terminal and run the following command:

```bash
pip install Flask chromadb flask_cors

fter installing the packages, make sure to run the Flask file on http://127.0.0.1:5000.

Step 2: Spin the Backend Server
Navigate to the backend directory in your terminal and run the following commands:

bash

npm install
This command installs the required Node.js dependencies.

Then, start the backend server using either of the following commands:

bash

npx nodemon
or

bash

node index.js
Step 3: Start the Frontend Server
Navigate to the frontend directory in your terminal and run the following commands:

bash

npm install
This command installs the required Node.js dependencies for the frontend.

Finally, start the frontend server using the following command:

bash

npm start
This will start the frontend server, and you should be able to access the application in your web browser.

Additional Notes
Make sure all the necessary ports are open and available on your system for running both the backend and frontend servers.
